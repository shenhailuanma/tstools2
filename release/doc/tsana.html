NAME

tsana - manual page for tsana 1.0.0  

SYNOPSIS

tsana [OPTION]...  

DESCRIPTION

'tsana' get TS packet from stdin, analyse, then send the result to stdout.  

OPTIONS

-lst
show PID list information, default option
-psi
show PSI tree information
-sec
show SI section data of cared <table>
-si
show SI section information of cared <table>
-outpsi
output PSI packet
-pcr
output PCR information of cared <pid>
-sys-rate
output system bit-rate
-psi-rate
output psi/si bit-rate
-prog-rate
output bit-rate of cared <prog>
-rate
output bit-rate of all PID
-pts
output PTS and DTS information of cared <pid>
-pes
output PES data of cared <pid>
-es
output ES data of cared <pid>
-alles
write ES data into different file by PID
-err
output all errors found
-dump
dump cared packet
-mono
disable colour effect, default: use colour to help read
-start <x>
analyse from packet(x), default: 0, first packet
-count <n>
analyse n-packet then stop, default: 0, no stop
-pid <pid>
set cared PID, default: any PID
-table <id>
set cared table, default: any table
-prog <prog>
set cared prog, default: any program
-iv <iv>
set cared interval(1ms-10,000ms), default: 1000ms
-h, --help
display this information
-v, --version
display my version
 

EXAMPLES

"bincat xxx.ts | tsana -err" -- report all error information
