NAME

catbin - manual page for catbin 1.0.0  

SYNOPSIS

catbin [OPTION] file [OPTION]  

DESCRIPTION

'catbin' read binary file, translate 0xXX to 'XY ' format, then send to stdout.  

OPTIONS

-bin
treat as common binary file, default
-ts
treat as TS file, sync and guess packet length at first
-mts
treat as MTS file, sync and guess packet length at first
-a, --addr
show data address at line head, default: do NOT show it
-d, --decaddr
dec address format, default: hex
-s, --seperate <,>
white space, any char except [0-9A-Fa-f], default: ' '
-w, --width <w>
w-byte per line, [1,10922], default: 188
-h, --help
display this information
-v, --version
display my version
 

EXAMPLES
